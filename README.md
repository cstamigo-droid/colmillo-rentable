# 🐶 Colmillo Rentable

Aprende a invertir de verdad, en 5 minutos al día. Estilo Duolingo, honesto, para LatAm.

Un camino corto y claro te lleva de "no sé nada" a "ya invertí mi primer índice y tengo el aporte
automático andando". Sin humo, sin promesas de hacerte rico rápido — paciencia + método.

## Probarlo
Es una web app estática. Ábrela en el navegador (funciona en el celular). Tu progreso se guarda en
tu propio navegador.

## Aviso
Contenido educativo, no asesoría financiera personalizada. Las decisiones de inversión son tuyas.
